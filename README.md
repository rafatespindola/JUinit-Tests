# REST API - Testes Unitários com JUnit

Projeto de estudos focado em testes unitários utilizando JUnit e Mockito em uma aplicação Spring Boot.

Desenvolvido como parte do curso de **Pós Graduação em Arquitetura e Desenvolvimento Java** da **FIAP**.

## Tecnologias

- Java
- Spring Boot
- JUnit 5
- Mockito
- AssertJ
- Maven
